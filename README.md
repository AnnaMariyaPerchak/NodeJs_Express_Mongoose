# NodeJs_Express_Mongoose

To start project:
  1. npm install
  2.npm start
  
HTTP-reguest,made by me :
  1.create new todo-action  http://localhost:3000/api/todo/add
  2.see all todo-actions  http://localhost:3000/api/todos
  3.update definite todo-action  http://localhost:3000/api/todo/_todoId_/update
  4.see definite todo-action  http://localhost:3000/api/todo/_todoId_
  5.delete definite todo-action  http://localhost:3000/api/todo/_todoId_/remove
  6.change field 'done' to 'true' in definite todo-action  http://localhost:3000/api/todo/_todoId_/done
  7.change field 'done' to 'false' in definite todo-action  http://localhost:3000/api/todo/_todoId_/undone
  
To 1 and 3 menu item you need to input new or updated dataset. You don`t need to do it with anothers.
